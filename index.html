Smart CY
Container Yard Management App

SMART CY is an Android-based mobile application optimized for tablets, specifically designed to streamline and automate container yard operations. Built using Flutter for the frontend, .NET for backend services, integrated with Firebase Authentication for secure user management, and utilizing MS SQL for reliable cloud-based database operations, SMART CY provides real-time container tracking, efficient transaction management, and increased yard visibility.

FSM Integrated Typing Game

An engaging culinary-themed typing game developed using Python and the Pygame library. The game utilizes Finite State Machines (FSMs) to handle dynamic state transitions, managing user interactions as players rapidly type food-related words to prepare dishes in a lively kitchen scenario.
